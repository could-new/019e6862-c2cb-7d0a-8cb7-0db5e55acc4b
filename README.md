# शोध रूपरेखा: शिक्षकों की पॉक्सो अधिनियम के प्रति जागरूकता

यह एक क्रॉस-प्लेटफ़ॉर्म फ़्लटर एप्लिकेशन है जिसे माध्यमिक स्तर के विद्यालयों के शिक्षकों की पॉक्सो (POCSO) अधिनियम के प्रावधानों के प्रति जागरूकता का प्रदर्शन करने वाले अध्ययन (शोध रूपरेखा/Synopsis) को प्रदर्शित करने के लिए डिज़ाइन किया गया है।

## सुविधाएँ

*   विस्तृत प्रस्तावना (Introduction): समस्या की पृष्ठभूमि, आवश्यकता और महत्व का वर्णन (2-3 पृष्ठ)।
*   अध्ययन के उद्देश्य (Objectives of the Study)।
*   परिकल्पना (Hypotheses)।
*   अनुसंधान विधि (Research Methodology)।
*   न्यादर्श (Sample)।
*   उपकरण (Tools)।
*   सांख्यिकीय प्रविधियाँ (Statistical Techniques)।

## टेक स्टैक

*   **फ़्रेमवर्क:** Flutter (Cross-platform)
*   **भाषा:** Dart
*   **UI/UX:** Material Design 3

## कैसे चलाएं (How to run)

1.  सुनिश्चित करें कि आपके सिस्टम पर Flutter स्थापित है।
2.  प्रोजेक्ट क्लोन करें या डाउनलोड करें।
3.  प्रोजेक्ट डायरेक्टरी में टर्मिनल खोलें और निर्भरताओं को स्थापित करने के लिए कमांड चलाएं:
    ```bash
    flutter pub get
    ```
4.  एप्लिकेशन को किसी भी उपलब्ध डिवाइस या एमुलेटर पर चलाएं:
    ```bash
    flutter run
    ```
